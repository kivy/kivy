The test case constructs a filter that uses feTile to tile the entire specified filter region. The tile consists of a red rectangle over a larger transparent rectangle. The red rectangle is created with feFlood and feOffset. There is also a semi-transparent blue rectangle that should exactly cover one of the tiled rectangles, creating a purple tile with a black stroke (4 tiles down and 3 across).

The rendered picture should match the reference image, except for possible variations in the labelling text (per CSS2 rules).

The test uses the 'rect' element, feTile, feFlood, feOffset, feMerge, fill style, stroke, font-family (Arial) and font-size properties.
